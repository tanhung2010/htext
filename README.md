### Text Utilities
    *upper case : [hteXt] -> [HTEXT]
    *lower case : [hteXt] -> [htext]
    *invert case : [hteXt] -> [HTExT]
    *proper case :  [hteXt] -> [Htext]
### Text line Utilities
    *sort A-Z: 
               [abc         [abc
               def     ->    bea
               bea]          def]
    *sort Z-A : 
               [abc         [def
               def     ->    bea
               bea]          abc]  
    *Remove Dupplicate lines
    *remove blank line
